# 📊 Netflix Streamlit Dashboard

---

## 📌 Overview

This project is an interactive **Streamlit dashboard** built to analyze Netflix content data and generate meaningful insights.

It allows users to explore data through filters and visualizations in a simple and interactive way.

---

## 🚀 Features

* Interactive filters (Genre, Country, Year)
* Content distribution analysis
* Genre and ratings insights
* Country-wise content trends
* Content growth over time

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas
* Plotly

---

## 📂 Files Included

* `app.py` → Main Streamlit application
* `requirements.txt` → Dependencies
* `netflix_cleaned.csv` → Dataset

---

## ▶️ How to Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
