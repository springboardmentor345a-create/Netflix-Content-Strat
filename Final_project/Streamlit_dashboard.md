## 📌 Overview

This project presents an interactive dashboard built using Streamlit to analyze Netflix content data.

It provides insights into:

* Content distribution
* Genre trends
* Ratings analysis
* Year-wise growth

---

## 🛠️ Tech Used

* Python
* Streamlit
* Pandas
* Plotly
