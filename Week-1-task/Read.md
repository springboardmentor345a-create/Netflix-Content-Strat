### Objective
This task focuses on loading and preprocessing the Netflix Movies and TV Shows dataset. The goal is to clean the dataset and prepare it for exploratory data analysis.

### Steps Performed
1. Loaded the Netflix dataset using pandas.
2. Checked missing values and handled them using imputation.
3. Removed duplicate records.
4. Normalized categorical features such as type, country, rating, and genres.
5. Saved the cleaned dataset for further analysis.

### Tools Used
- Python
- Pandas
- Jupyter Notebook

### Output
A cleaned dataset file netflix_cleaned.csv is generated for further milestones.
